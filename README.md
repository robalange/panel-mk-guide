# panel-mk-guide

## 项目简介

`panel-mk-guide` 是一个用于归档和发布多个独立 HTML 页面的仓库。本仓库不针对任何特定的域名或网站，旨在以静态页面的形式整理和展示一系列指南、说明或工具页面。

## 目录结构

```
panel-mk-guide/
├── README.md          # 本文件
├── index.html         # 仓库首页（可选）
├── pages/             # 存放独立 HTML 页面
│   ├── example1.html
│   ├── example2.html
│   └── ...
└── assets/            # 静态资源（CSS、JS、图片等）
    ├── css/
    ├── js/
    └── images/
```

- `pages/`：主要目录，所有独立的 HTML 页面归档于此。
- `assets/`：存放页面所需的公共资源文件。

## 页面归档说明

本仓库中的每个 HTML 页面均为独立、自包含的文档，可直接在浏览器中打开查看。页面内容涵盖各类指南、操作说明或参考信息，不依赖于外部服务或特定域名。

如需添加新页面，请遵循以下原则：

- 页面文件放置于 `pages/` 目录下。
- 文件名使用小写英文，单词间以连字符 `-` 分隔。
- 页面内不包含指向具体域名、网站或营销内容的链接。
- 如有公共资源引用，请统一放置在 `assets/` 目录中。
